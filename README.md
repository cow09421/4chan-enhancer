# 4chan 自訂圖庫增強版 (4chan-enhancer)

這是一個專為 4chan / 4channel 討論串設計的 Manifest V3 Chrome 擴充套件，提供更佳的圖片瀏覽體驗。

## 功能特色
- 圖片網格檢視（根據視窗寬度動態調整 2~6 欄）。
- 支援 WEBM、MP4 與 GIF 媒體播放與標示。
- 內建燈箱檢視（Lightbox），支援鍵盤左右鍵翻頁及預載下一張圖。
- 圖片分頁顯示，支援自訂每頁數量並記憶設定（12 / 24 / 48 筆）。
- 一鍵批量下載當前討論串的所有靜態圖片。
- 無縫整合於 4chan 原生導覽列中。

## 安裝與執行 (開發者模式)
1. 下載或 Clone 此專案到本地。
2. 開啟 Google Chrome 或基於 Chromium 的瀏覽器。
3. 進入擴充功能管理頁面（網址列輸入 `chrome://extensions/`）。
4. 點擊右上角開啟「**開發人員模式**」。
5. 點擊左上角「**載入未封裝項目**」（Load unpacked）。
6. 選擇本專案資料夾 `4chan-enhancer`。
7. 前往任何一個 4chan 討論串頁面，即可在頂部導覽列看到 `[圖片網格]` 按鈕。

## 專案結構
- `manifest.json`: 擴充套件設定與權限配置。
- `content.js`: 主要邏輯腳本，負責抓取圖片、生成網格、綁定事件與處理互動。
- `styles.css`: 擴充套件的樣式與 UI 設計。
